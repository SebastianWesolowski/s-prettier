Configuration for prettier

## Install

```bash
npm install s-prettier
```

## Usage

use package as configuration for prettier in package.json

```json
{
    "prettier": "s-prettier",
    "devDependencies": {
        "prettier": "^3.2.5",
        "s-prettier": "^1.0.1"
    }
}
```
